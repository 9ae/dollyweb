Keyboard Controls
Rotate about x-axis: [↑] [↓]
Rotate about y-axis: [←] [→]
Rotate about z-axis: [A] [S]
Zoom in/out: [Z] [X]
Release for any selection: [SPACE]
The model on the right, indicates which keys will activate which joints

Mouse Controls Click on the part you want to move, and drag [keep mouse button pressed] your mouse about to rotate the joints. This movement function is still slightly shakey and needs work. I recommend using the keys to rotate joitns for now.
